# ProductionShipwreckArrivalCharacterV73

V73 交付同一 V61/V64 幸存者的 8 帧“船尾抓附—脱手—游泳接管”动作。F00-F03 后手固定在 V72 艉部抓点；F04 脱手；F07 确定性继承 V64 游泳端点。所有帧保留完整双腿和双靴，人物必须放在 V72 后层残骸与前层船舷之间，再由连续海水遮挡水下部分。

`Runtime/High` 是 1024 逻辑画布、512 PPU 的配准权威；`Runtime/Balanced` 是 768 逻辑画布、384 PPU 的轻量候选。实际 PNG 均紧裁，并通过 Unity 自定义 Pivot 保持共同逻辑原点。当前包没有修改 Scene、Prefab、Resources、Animator、控制器或权威运行清册。
