# Tide V72 同船海难来路

V72 把开场主残骸改为 V39/V46 同一艘单桅叠板船的断裂态。六个 owner 共用 ocean root，运行时只插值 Transform。

- `runtime-contract.json`：三档资源、六层锚点、三阶段和人物到达锚点。
- `Runtime/`：UHD / High / Balanced 可原子切换的透明 Sprite。
- `QA/`：同船身份、分层、三阶段、16:9 / 4:3 与 GIF 证据。
- `Source/`：imagegen 生成与色键去除来源，仅作生产溯源，禁止运行接入。

V55 不被替换，它继续负责后续短航的稀疏漂流物。V72 未连接 Unity、场景、Prefab、Catalog 或权威运行清册。
