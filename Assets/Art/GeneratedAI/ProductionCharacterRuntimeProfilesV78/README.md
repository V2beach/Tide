# Tide V78 人物全动作双运行档

V78 不重画人物。V61 的 136 帧、21 个动作族仍是姿势与 S2 材质母版；本包只提供 `High` 和 `Balanced` 两档预乘 Alpha 派生图。

- `High`：最长边不超过 1024，供近景或审片。
- `Balanced`：最长边不超过 512，推荐第一切片常规 Game View。
- 两档都保留完整透明画布、归一化 Pivot 和世界画布尺寸，不裁腿、不挪脚、不改手点。
- 运行必须按动作族加载，切换时最多短暂并存两个动作族；禁止一次常驻全部 136 帧。
- `QA` 只用于验收，禁止写入 Catalog、Scene 或 Prefab。
